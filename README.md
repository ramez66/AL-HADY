
## المميزات

1. **تصميم عصري متجاوب** مع جميع أحجام الشاشات
2. **واجهة عربية كاملة** مع دعم اتجاه RTL
3. **نظام سلة تسوق** متكامل مع localStorage
4. **أقسام متنوعة**: المنتجات، العروض، آراء العملاء، الأسئلة الشائعة
5. **صفحات متعددة**: الرئيسية، المنتجات، سلة التسوق، اتصل بنا
6. **إمكانيات خاصة**:
   - تصفية المنتجات
   - إضافة/حذف المنتجات من السلة
   - حساب المجموع تلقائياً
   - نموذج تواصل

## التقنيات المستخدمة

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6+)
- Font Awesome للرموز
- تصميم متجاوب (Responsive Design)

## كيفية الاستخدام

1. قم بتنزيل جميع الملفات
2. افتح `index.html` في متصفحك
3. للمراجعة:
   - تصفح المنتجات من الصفحة الرئيسية
   - أضف منتجات إلى سلة التسوق
   - تابع إلى صفحة سلة التسوق لإتمام الشراء
   - استخدم نموذج التواصل للاتصال

## الصور

يجب إضافة الصور في المجلدات التالية:
- `assets/images/` - للصور العامة
- `assets/images/products/` - لصور المنتجات
- `assets/images/icons/` - للأيقونات

## الملفات الإضافية المطلوبة

1. `logo.png` في `assets/images/`
2. `hero-bg.jpg` في `assets/images/`
3. صور المنتجات في `assets/images/products/`:
   - `olive-oil.jpg`
   - `rice.jpg`
   - `coffee.jpg`
   - `dates.jpg`
   - `supermarket.jpg`
